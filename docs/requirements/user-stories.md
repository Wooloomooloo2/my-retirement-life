# User Stories

User stories for My Retirement Life, organised by feature area. Each story follows the format:

> **As a** [type of user], **I want to** [do something], **so that** [I get some value].

Stories marked ✅ are in scope for MVP. Stories marked 🔮 are post-MVP.

---

## Personal Profile

- ✅ As a user, I want to enter my age and target retirement age, so that the app knows my planning horizon.
- ✅ As a user, I want to enter my current annual net income, so that the app can model my savings rate.
- ✅ As a user, I want to set a life expectancy, so that the projection runs to a meaningful end point.
- ✅ As a user, I want to set my base currency, so that all figures are displayed consistently.
- ✅ As a user, I want to update my profile at any time, so that my projection stays current.
- 🔮 As a user, I want to model a partner's profile alongside my own, so that I can plan as a couple.

---

## Cash Accounts

- ✅ As a user, I want to add one or more cash accounts with a name, balance, and interest rate, so that my savings are reflected in the projection.
- ✅ As a user, I want to distinguish between saving accounts and spending accounts, so that the model knows which funds are available for retirement spending.
- ✅ As a user, I want to edit or delete a cash account, so that I can keep my data accurate.
- 🔮 As a user, I want to add investment accounts (stocks, funds, bonds), so that my full portfolio is modelled.
- 🔮 As a user, I want to add accounts in different currencies, so that my international assets are included.

---

## Income

- ✅ As a user, I want to enter my current annual net income, so that my accumulation phase is correctly modelled.
- ✅ As a user, I want to apply an annual percentage increase or decrease to my income, so that pay rises or a move to part-time work are reflected.
- ✅ As a user, I want to set a retirement year after which employment income stops, so that the drawdown phase begins correctly.
- 🔮 As a user, I want to add a state pension with a start age and annual amount, so that my retirement income is more complete.
- 🔮 As a user, I want to add private pension income with a start age and annual amount, so that defined benefit pensions are reflected.
- 🔮 As a user, I want to add other income sources (rental income, part-time work in retirement), so that my full income picture is captured.

---

## Budget

- ✅ As a user, I want to enter mandatory spending items with a name and annual amount, so that essential costs are deducted from my projection.
- ✅ As a user, I want to enter discretionary spending items, so that lifestyle costs are included.
- ✅ As a user, I want to apply an annual percentage change to any budget line, so that inflation or lifestyle changes are modelled.
- ✅ As a user, I want to enter loan repayments with an end date, so that my budget automatically reduces when loans are paid off.
- 🔮 As a user, I want my budget to automatically change at retirement (e.g. commuting costs drop), so that I don't have to manage this manually.

---

## Life Events

- ✅ As a user, I want to add a one-off large expenditure at a specific future year, so that known costs are reflected in my projection.
- ✅ As a user, I want to give each life event a name and description, so that I remember what it represents.
- ✅ As a user, I want to edit or delete life events, so that my plan stays up to date.
- 🔮 As a user, I want to add a house purchase or sale as a life event, so that property equity is reflected.
- 🔮 As a user, I want to add a windfall (inheritance, bonus, asset sale) at a future year, so that expected receipts are included.
- 🔮 As a user, I want to model moving abroad as a life event, so that currency and cost-of-living changes are reflected.
- 🔮 As a user, I want to model caring responsibilities as a life event, so that the financial impact of caring for family is included.

---

## Retirement Projection

- ✅ As a user, I want to see a year-by-year burndown chart of my cash position, so that I can visualise my retirement trajectory.
- ✅ As a user, I want to see the year my cash runs out (if applicable), clearly marked on the chart, so that I understand the risk.
- ✅ As a user, I want to see a confidence score indicating whether my plan is on track, so that I get a quick summary without reading the full chart.
- ✅ As a user, I want the projection to update immediately when I change any input, so that I can see the effect of my decisions in real time.
- 🔮 As a user, I want to compare two scenarios side by side, so that I can evaluate the impact of different choices.
- 🔮 As a user, I want a probabilistic confidence score based on market variability, so that I understand the range of possible outcomes.

---

## General

- ✅ As a user, I want my data to be stored locally on my machine, so that my financial information stays private.
- ✅ As a user, I want the app to open in my browser with a single click, so that I don't need technical knowledge to use it.
- 🔮 As a user, I want to export my data as a CSV or PDF report, so that I can share it with a financial adviser.
- 🔮 As a user, I want to import data from My Financial Life, so that my actual spending feeds into my retirement plan.
