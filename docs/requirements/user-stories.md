# User Stories

User stories for My Retirement Life, organised by feature area. Each story follows the format:

> **As a** [type of user], **I want to** [do something], **so that** [I get some value].

Stories marked ✅ are complete in MVP. Stories marked 🔮 are post-MVP.

---

## Personal Profile

- ✅ As a user, I want to enter my age and target retirement age, so that the app knows my planning horizon.
- ✅ As a user, I want to set a life expectancy, so that the projection runs to a meaningful end point.
- ✅ As a user, I want to set my base currency, so that all figures are displayed consistently.
- ✅ As a user, I want to update my profile at any time, so that my projection stays current.
- 🔮 As a user, I want to model a partner's profile alongside my own, so that I can plan as a couple.

---

## Income

- ✅ As a user, I want to add multiple income sources, so that my full income picture is captured — not just employment.
- ✅ As a user, I want to enter my current annual net income, so that my accumulation phase is correctly modelled.
- ✅ As a user, I want to apply an annual percentage increase or decrease to any income source, so that pay rises, inflation, or changes over time are reflected.
- ✅ As a user, I want to set a start year on an income source, so that future income (e.g. a pension starting at 67) is correctly modelled.
- ✅ As a user, I want to set an end year on an income source, so that income that stops at a specific point (e.g. employment stopping at retirement) is correctly modelled.
- ✅ As a user, I want to leave the end year blank on an income source, so that income that continues indefinitely (e.g. rental income) is correctly modelled.
- ✅ As a user, I want to classify income by type (employment, property, investment etc.), so that my income picture is organised and extensible.
- ✅ As a user, I want to edit or delete income sources at any time, so that my projection stays accurate as my circumstances change.
- 🔮 As a user, I want to add a state pension with a start age and jurisdiction-specific rules, so that my state entitlement is correctly modelled.
- 🔮 As a user, I want to add defined benefit pension income, so that guaranteed employer pension is reflected.
- 🔮 As a user, I want to model income in a foreign currency, so that international income sources are correctly converted.

---

## Cash Accounts

- ✅ As a user, I want to add one or more cash accounts with a name, balance, balance date, and interest rate, so that my savings are reflected in the projection.
- ✅ As a user, I want to classify accounts by type (current, savings, fixed term, tax-advantaged), so that my accounts are organised.
- ✅ As a user, I want to add accounts in any currency with a manual exchange rate, so that international savings are included in my projection.
- ✅ As a user, I want to record the date my balance was captured, so that the projection calculates from an accurate starting point.
- ✅ As a user, I want to edit or delete a cash account, so that I can keep my data accurate.
- 🔮 As a user, I want to add investment accounts (stocks, funds, bonds), so that my full portfolio is modelled.
- 🔮 As a user, I want to set drawdown priority on accounts, so that I can control which accounts are drawn down first in retirement.
- 🔮 As a user, I want interest and income to be credited to specific accounts, so that account balances grow accurately.

---

## Budget

- ✅ As a user, I want to enter mandatory spending items, so that essential costs are deducted from my projection.
- ✅ As a user, I want to enter discretionary spending items, so that lifestyle costs are included.
- ✅ As a user, I want to enter spending at any frequency (weekly, monthly, annually etc.), so that I don't have to convert amounts myself.
- ✅ As a user, I want to see the annual equivalent of any spending entry, so that I can verify my inputs are correct.
- ✅ As a user, I want to apply an annual percentage change to any budget line, so that inflation or lifestyle changes are modelled.
- ✅ As a user, I want to enter loan repayments with an end year, so that my budget automatically reduces when loans are paid off.
- 🔮 As a user, I want to set different growth rates for different periods on a budget line, so that I can model spending that changes dramatically at a specific point (e.g. holidays increasing at retirement, then reducing at 75).
- 🔮 As a user, I want my budget to automatically change at retirement, so that costs like commuting drop without manual adjustment.

---

## Life Events

- ✅ As a user, I want to add a one-off large expenditure at a specific future year, so that known costs are reflected in my projection.
- ✅ As a user, I want to add a windfall (negative amount) at a future year, so that expected receipts like inheritance are included.
- ✅ As a user, I want to give each life event a name, type, and notes, so that I remember what it represents.
- ✅ As a user, I want to edit or delete life events, so that my plan stays up to date.
- 🔮 As a user, I want to add a house purchase or sale as a life event, so that property equity changes are reflected.
- 🔮 As a user, I want to model moving abroad as a life event, so that currency and cost-of-living changes are reflected.
- 🔮 As a user, I want to model caring responsibilities as a life event, so that the financial impact of caring for family is included.

---

## Retirement Projection

- ✅ As a user, I want to see a year-by-year stacked chart of my spending and balance, so that I can visualise my retirement trajectory clearly.
- ✅ As a user, I want to see the year my cash runs out (if applicable), so that I understand the risk.
- ✅ As a user, I want to see a confidence score indicating whether my plan is on track, so that I get a quick summary.
- ✅ As a user, I want to see my balance at retirement and at life expectancy, so that I understand my financial trajectory.
- ✅ As a user, I want to set a default inflation rate assumption, so that budget lines without a growth rate are realistically modelled.
- 🔮 As a user, I want to compare two scenarios side by side, so that I can evaluate the impact of different choices.
- 🔮 As a user, I want a probabilistic confidence score based on market variability, so that I understand the range of possible outcomes.

---

## Dashboard

- ✅ As a user, I want to see my key retirement metrics at a glance on the dashboard, so that I get an immediate sense of where I stand.
- ✅ As a user, I want to see a mini balance trajectory chart on the dashboard, so that I can see my overall direction without opening the full projection.
- ✅ As a user, I want quick links to each data entry section, so that I can navigate efficiently.

---

## General

- ✅ As a user, I want my data to be stored locally on my machine, so that my financial information stays private.
- ✅ As a user, I want the app to open in my browser with a single click, so that I don't need technical knowledge to use it.
- ✅ As a user, I want to see my initials in the app, so that it feels personalised.
- 🔮 As a user, I want to export my data as a CSV or PDF report, so that I can share it with a financial adviser.
- 🔮 As a user, I want to import data from My Financial Life, so that my actual spending feeds into my retirement plan.
- 🔮 As a user, I want the app to be available as a single downloadable file, so that installation is trivial.
